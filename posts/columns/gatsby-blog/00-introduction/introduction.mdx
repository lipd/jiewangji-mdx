本专栏主要通过使用 Gatsby 从零搭建一个博客系统，使读者了解 Gatsby 在实际使用中的诸多细节。我们还将在项目中使用腾讯云 Serverless 将网站轻松部署在国内的网络环境中。

## Gatsby 是什么？

Gatsby 是一个基于 react 生态的现代网站生成工具，只要你对 react 有一定了解，你就可以通过 Gatsby 快速搭建一个完整的网站，并且迅速将其部署上线。从功能上来看 Gatsby 更加接近于 Jekyll、VuePress、Hexo 之类的静态网站生成器，而不是 WordPress 这样的内容管理系统（CMS），但 Gatsby 又比传统的静态网站生成器更加强大，并且拥有一个更加良好的生态。

## Gatsby 适合哪些业务场景？

- 博客网站
- 为企业或者产品制作展示页面
- 电子商务网站

## 本专栏适合哪些读者？

本专栏主要面向有一定 JavaScript 的使用经验（最好了解 ES6 的基本语法）并且对 react 有最基本的了解（会使用 react 编写一个基本组件）的同学。你可以将本专栏作为你学习 Gatsby 的起点，也可以将本专栏视为官方文档的补充资料。

本文将会涉及到 GraphQL 与 Serverless Framework 的相关知识，但你不需要现在就了解和掌握他们。GraphQL 相关的知识我们可以在编写项目的过程中进行学习；
而对于 Serverless Framework，在这个项目中我们只使用其最基本的功能。不过建议你提前注册一个腾讯云账号，方便我们之后进行部署工作。
